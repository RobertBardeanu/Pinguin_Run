# Penguin Run

Ein endloser Side-Scroller, in dem du einen Pinguin durch eine verschneite Stadt steuerst. Weiche Robben, Netzen und Bäumen aus, sammle Items und überlebe so lange wie möglich.

## Funktionen
- Endloses Side-Scroller-Gameplay
- Hindernissen ausweichen
- Items sammeln
- Steigende Geschwindigkeit
- Pixeliger, cartoonartiger Winterstil

## Start
Das Projekt in einer Java-IDE öffnen und starten.

## Technologien
Java
